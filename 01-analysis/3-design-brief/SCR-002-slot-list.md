# SCR-002 · Список слотов

> Список предстоящих мастер-классов с фильтрами. Стартовый экран авторизованной зоны.

**Источники:** FR-9, FR-38, FR-9a, US-2, US-3

## Навигация

| Вход | Выход | Тип |
|:-----|:------|:----|
| SCR-001 (вход), или Таб-бар «Занятия» | SCR-003 (карточка), BS-001 (фильтры) | Push / Bottom Sheet |

## Компоненты

### 1. Хедер
- Подзаголовок: адрес мастерской (из `workshop_address` первого слота или дефолт).
- Дата: сегодняшняя дата в формате «Сегодня, 5 июля».

### 2. Кнопка фильтров
- Иконка + текст «Фильтры».
- Индикатор (badge/dot) когда фильтры активны.
- Тап → BS-001.

### 3. Карточка слота
- **Иконка программы:** 🎨 лепка / 🏺 круг (визуальное различие).
- **Название программы:** крупный текст.
- **Мастер + время:** `🧑‍🎨 {name} · {start_at, HH:MM}`.
- **Места + цена:** `💺 {free_seats} из {total_seats} мест · {price} ₽`.
- **CTA:**
  - `free_seats > 0` → кнопка «Записаться», активная.
  - `free_seats = 0` → кнопка «Мест нет», disabled.
  - `status = cancelled` → кнопка «Занятие отменено», disabled.

### 4. Пустое состояние
- Иллюстрация + «Пока нет доступных занятий» + «Проверьте расписание позже».

## Состояния

| Состояние | Отображение |
|:--|:--|
| Loading | Скелетон карточек (3 шт.) |
| Content | Список с группировкой по датам |
| Empty | Заглушка «Пока нет доступных занятий» |
| Error | Заглушка + «Обновить» (текст из §6 foundations) |

## Технические детали

- **API:** `GET /slots?date_from=now&date_to=now+7d&limit=20&offset=0`
- **Сортировка:** по `start_at` по возрастанию.
- **Дефолт:** `only_available=false` (показываются все слоты).
- **Pull-to-refresh:** поддерживается.
