# SCR-004 · Оформление записи

> Экран выбора числа мест и инструментов перед подтверждением брони.

**Источники:** FR-10, FR-11, FR-12, FR-13, FR-14, FR-15, FR-45, US-4

## Навигация

| Вход | Выход | Тип |
|:-----|:------|:----|
| SCR-003 (карточка слота, тап «Записаться») | BS-002 (успех) / SCR-003 (ошибка) | Bottom Sheet |

## Компоненты

### 1. Заголовок
- Стрелка назад + «Оформление записи».
- Инфо слота: программа, мастер, время, дата.

### 2. Счётчик мест
- Степпер [−] [+] с числом посередине.
- Минимум: 1.
- Максимум: `min(free_seats, program.capacity_cap, 3)` — от API (`max_seats`).
- При достижении лимита кнопка [+] блокируется.

### 3. Выбор инструментов на каждое место
- Радио-кнопки для каждого участника:
  - «Свои инструменты» (бесплатно).
  - «Прокатный набор» (+ `rental_price` ₽).
- Если `free_rental_kits` исчерпан — опция «Прокатный набор» disabled с пояснением «Нет в наличии».

### 4. Расчёт цены
- Показывает `price_total` из ответа API (не пересчитывает локально).
- Детализация: `N мест × X ₽ + M прокатов × Y ₽`.

### 5. Нижний CTA
- «Подтвердить запись» — активен если валидно.
- При тапе: `POST /bookings` → ожидание → BS-002 (успех) или ошибка.

## Обработка ошибок

| Ошибка | Отображение |
|:--|:--|
| 409 Conflict (мест нет) | Снек «К сожалению, мест уже нет. Выберите другой слот.» |
| 409 Conflict (проката нет) | Снек «Прокатных наборов не хватает. Выберите свои инструменты.» |
| 410 Gone (слот отменён) | Снек «Занятие отменено мастерской.» → возврат к SCR-002 |
| Сеть | Снек из §6 foundations |

## Технические детали

- **API:** `POST /bookings` с `slot_id` и массивом `seats`.
- **Валидация:** перед отправкой проверяет, что `seats.length ≤ max_seats` и `rental_count ≤ free_rental_kits`.
