# Карта навигации проекта (Navigation Map)

> Граф переходов между экранами приложения «Глина». Аналог ER-диаграммы для UI-слоя.

## Визуальная схема

```mermaid
flowchart TD
    subgraph "Незарегистрированная зона (НЗ)"
        LAUNCH["🚀 Запуск приложения"]
        SCR001["SCR-001<br/>Регистрация / Вход"]
        LAUNCH --> SCR001
    end

    SCR001 -->|"OTP подтверждён"| SCR002

    subgraph "Авторизованная зона (АЗ) — Таб-бар"
        direction TB

        subgraph "Вкладка: Занятия"
            SCR002["SCR-002<br/>Список слотов"]
            BS001["BS-001 🡅<br/>Фильтры"]
            SCR003["SCR-003<br/>Карточка слота"]
            SCR004["SCR-004<br/>Оформление записи"]
            BS002["BS-002 🡅<br/>Подтверждение записи"]

            SCR002 <-->|"Фильтры"| BS001
            SCR002 -->|"Тап по слоту"| SCR003
            SCR003 -->|"Записаться"| SCR004
            SCR004 -->|"Подтвердить"| BS002
            BS002 -->|"Мои бронирования"| SCR005
            BS002 -->|"К занятиям"| SCR002
        end

        subgraph "Вкладка: Мои записи"
            SCR005["SCR-005<br/>Мои бронирования"]
            SCR006["SCR-006<br/>Детали брони"]
            BS003["BS-003 🡅<br/>Подтверждение отмены"]

            SCR005 -->|"Тап по брони"| SCR006
            SCR006 -->|"Отменить"| BS003
            BS003 -->|"Подтвердить"| SCR006
        end

        subgraph "Вкладка: Профиль"
            SCR007["SCR-007<br/>Профиль клиента"]

            SCR007 -->|"Выйти / Удалить"| SCR001
        end
    end

    %% Таб-бар переключение
    SCR002 -.->|"🔄 Таб: Мои записи"| SCR005
    SCR005 -.->|"🔄 Таб: Профиль"| SCR007
    SCR007 -.->|"🔄 Таб: Занятия"| SCR002
    SCR005 -.->|"🔄 Таб: Занятия"| SCR002

    %% Декоративные стили
    style LAUNCH fill:#e1f5fe
    style SCR001 fill:#fff3e0
    style BS001 fill:#f3e5f5
    style BS002 fill:#e8f5e9
    style BS003 fill:#ffebee
```

## Глубина навигации

| Сценарий | Путь | Глубина (экранов) |
|:--|:--|:--|
| Вход | SCR-001 → SCR-002 | 2 |
| Запись | SCR-002 → SCR-003 → SCR-004 → BS-002 | 4 (3 + шторка) |
| Отмена | SCR-005 → SCR-006 → BS-003 | 3 (2 + шторка) |
| Фильтрация | SCR-002 ↔ BS-001 | 1 + шторка |

> **Требование (P2):** От списка до подтверждения записи — **≤ 3 экранов**. Путь
> SCR-002 → SCR-003 → SCR-004 → BS-002 соответствует (2 перехода + финальная шторка).
