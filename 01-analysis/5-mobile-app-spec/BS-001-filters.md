# Фильтры слотов

**ID:** BS-001  
**Тип:** Bottom Sheet  
**Домен:** 02. Слоты  
**Приоритет:** High  
**Зона:** АЗ

## Обзор

Шторка фильтрации слотов по дате, программе (лепка/круг) и мастеру.

## Навигация

| Вход | Выход |
|------|-------|
| SCR-002 (тап «Фильтры») | SCR-002 (Применить / Сбросить) |

## Используемые запросы

### listMasters

**GET `/masters`** — загружается при открытии шторки.

## Фильтры

| Фильтр | Параметр API | Тип |
|--------|-------------|-----|
| Дата | `date_from`, `date_to` | date picker |
| Программа | `program_type` | множественный выбор (`handbuilding`, `wheel`) |
| Мастер | `master_id` | единичный выбор из справочника |

## Критерии приёмки

| ID | Критерий |
|----|----------|
| AC-001 | Множественный выбор программ |
| AC-002 | Справочник мастеров из API |
| AC-003 | «Применить» → обновление SCR-002 |
| AC-004 | «Сбросить» → сброс всех фильтров |
