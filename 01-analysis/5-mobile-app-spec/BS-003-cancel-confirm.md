# Подтверждение отмены

**ID:** BS-003  
**Тип:** Bottom Sheet  
**Домен:** 03. Бронирования  
**Приоритет:** High  
**Зона:** АЗ

## Обзор

Шторка подтверждения отмены брони с объяснением правила 2 часов.

## Навигация

| Вход | Выход |
|------|-------|
| SCR-006 (тап «Отменить бронь») | SCR-006 (подтверждение/отмена) |

## Критерии приёмки

| ID | Критерий |
|----|----------|
| AC-001 | Показывается название и дата отменяемой записи |
| AC-002 | Текст правила отмены (2 часа) — из foundations |
| AC-003 | «Да, отменить» → `DELETE /bookings/{id}` |
| AC-004 | «Назад» → закрытие шторки, бронь не меняется |
