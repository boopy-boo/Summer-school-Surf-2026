## Обзор

Стартовый экран авторизованной зоны. Показывает список мастер-классов на ближайшие 7 дней с группировкой по дате. Поддерживает фильтрацию и pull-to-refresh.

## Навигация

### Входящая

| Источник | Триггер |
|----------|---------|
| SCR-001 | Успешный вход |
| Таб-бар | Вкладка «Занятия» |

### Исходящая

| Назначение | Триггер |
|------------|---------|
| SCR-003 | Тап по карточке слота |
| BS-001 | Тап «Фильтры» |

## Входные данные

| Название | Тип | Описание |
|----------|-----|----------|
| `filters` | Состояние экрана | Активные фильтры (дата, программа, мастер) |
| `slots` | Кэш / API | Список слотов |

## Инициализация

При открытии: `GET /slots` с дефолтными параметрами (`date_from=now`, `date_to=now+7d`).

### Диаграмма загрузки

```mermaid
flowchart LR
    Start([onEnter]) --> Loading
    Loading --> Content: 200 OK
    Loading --> Empty: пустой список
    Loading --> Error: ошибка
```

## Используемые запросы

### listSlots

**GET `/slots`**  
**Триггер:** Открытие экрана, pull-to-refresh, применение фильтров

| Параметр | Тип | Источник | Дефолт |
|----------|-----|----------|--------|
| `date_from` | datetime | Текущее время | now |
| `date_to` | datetime | now + 7 дней | now+7d |
| `program_type` | array | Фильтры | — |
| `master_id` | array | Фильтры | — |
| `only_available` | boolean | Фильтры | false |
| `limit` | int | Пагинация | 20 |
| `offset` | int | Пагинация | 0 |

**Обработка ответа:**

| Результат | Условие | UI-реакция |
|-----------|---------|------------|
| Успех | 200, items > 0 | Список с группировкой по дате |
| Пусто | 200, items = 0 | Empty state «Пока нет доступных занятий» |
| Ошибка | 4xx/5xx/сеть | Error state + «Обновить» |

## Элементы экрана

| Элемент | Описание |
|---------|----------|
| Адрес мастерской | Из первого слота или дефолт |
| Дата | «Сегодня, ДД месяц» |
| Кнопка фильтров | Индикатор активных фильтров (badge) |
| Карточка слота | Программа, мастер, время, места, цена, CTA |
| Таб-бар | 3 вкладки |

### Карточка слота

| Поле | Источник |
|------|----------|
| Иконка программы | `program.type` (🎨/🏺) |
| Название | `program.name` |
| Мастер + время | `master.name`, `start_at` |
| Места | `free_seats` / `total_seats` |
| Цена | `price` ₽ |
| CTA | Активен если `free_seats > 0` и `status != cancelled` |

## Состояния экрана

| Состояние | Отображение |
|-----------|-------------|
| Loading | Скелетон (3 карточки) |
| Content | Список с группировкой по дате |
| Empty | «Пока нет доступных занятий» |
| Error | «Не удалось загрузить» + «Обновить» |
| Фильтры активны | Индикатор на кнопке, счётчик результатов |

## Действия пользователя

| Действие | Результат |
|----------|-----------|
| Тап по карточке | Переход на SCR-003 |
| Тап «Фильтры» | Открытие BS-001 |
| Pull-to-refresh | Перезагрузка списка |
| Скролл вниз | Пагинация (load more) |

## Критерии приёмки

### Позитивные

| ID | Критерий |
|----|----------|
| AC-001 | При открытии загружаются слоты на 7 дней, отсортированные по времени |
| AC-002 | Карточка с `free_seats=0` показывает «Мест нет» (disabled) |
| AC-003 | Pull-to-refresh обновляет список |

### Негативные

| ID | Критерий |
|----|----------|
| AC-N01 | При отсутствии слотов — empty state с понятным сообщением |
| AC-N02 | При ошибке сети — кнопка «Обновить», не пустой экран |
