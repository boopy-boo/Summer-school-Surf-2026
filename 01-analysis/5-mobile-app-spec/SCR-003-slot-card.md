# Карточка слота

**ID:** SCR-003  
**Тип:** Экран  
**Домен:** 02. Слоты  
**Приоритет:** Critical  
**Зона:** АЗ

## Обзор

Детальная информация о выбранном мастер-классе: программа, мастер, время, адрес на карте, цена, доступность мест и инструментов. Отсюда начинается запись.

## Навигация

| Вход | Выход |
|------|-------|
| SCR-002 (тап по карточке) | SCR-004 (тап «Записаться») |

## Используемые запросы

### getSlot

**GET `/slots/{slotId}`**

| Параметр | Тип | Источник |
|----------|-----|----------|
| `slotId` | uuid | Передан из SCR-002 |

**Обработка ответа:**

| Результат | Условие | UI-реакция |
|-----------|---------|------------|
| Успех | 200 | Отображение всех данных слота |
| 404 | Слот не найден | Error state + «Вернуться к списку» |

## Элементы экрана

| Элемент | Источник | Примечание |
|---------|----------|------------|
| Название программы | `program.name` | — |
| Тип программы | `program.type` | Лепка / Круг |
| Описание | `program.description` | — |
| Дата, время | `start_at` | Локальная зона |
| Длительность | `program.duration_minutes` | — |
| Мастер | `master.name` | — |
| Карта | `workshop_coordinates` | Статичный превью Яндекс |
| Адрес | `workshop_address` | Fallback при недоступности карты |
| Свободно мест | `free_seats` / `total_seats` | — |
| Свободно проката | `free_rental_kits` | — |
| Цена места | `price` ₽ | — |
| Цена проката | `rental_price` ₽ | — |
| CTA «Записаться» | — | Disabled если `free_seats=0` или `status=cancelled` |

## Критерии приёмки

| ID | Критерий |
|----|----------|
| AC-001 | При открытии загружаются полные данные слота |
| AC-002 | Карта показывает статичный превью с пином адреса |
| AC-003 | При недоступности карты — текстовый fallback |
| AC-004 | CTA disabled если мест нет или слот отменён |
