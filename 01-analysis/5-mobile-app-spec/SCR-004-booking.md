# Оформление записи

**ID:** SCR-004  
**Тип:** Экран  
**Домен:** 03. Бронирования  
**Приоритет:** Critical  
**Зона:** АЗ

## Обзор

Выбор числа мест (1–3) и инструментов (свои / прокат) для каждого участника. Итоговая цена отображается из API.

## Навигация

| Вход | Выход |
|------|-------|
| SCR-003 (тап «Записаться») | BS-002 (успех) |

## Используемые запросы

### createBooking

**POST `/bookings`**

| Параметр | Тип | Источник |
|----------|-----|----------|
| `slot_id` | uuid | Передан из SCR-003 |
| `seats` | array | Выбор пользователя |

**Обработка ответа:**

| Результат | Условие | UI-реакция |
|-----------|---------|------------|
| Успех | 201 | BS-002 (подтверждение) |
| 409 | Мест нет | Снек «Мест уже нет» → SCR-002 |
| 409 | Проката нет | Снек «Проката не хватает» |
| 410 | Слот отменён | Снек «Занятие отменено» → SCR-002 |
| Сеть | — | Снек из foundations |

## Элементы экрана

| Элемент | Описание | Валидация |
|---------|----------|-----------|
| Степпер мест | [−] [+] | min 1, max = min(free_seats, 3) |
| Радио инструменты | Свои / Прокат на каждое место | Прокат disabled если `free_rental_kits` исчерпан |
| Итог | `price_total` | От API, не пересчитывается локально |
| CTA | «Подтвердить запись» | Disabled при превышении лимитов |

## Связанные требования

| ID | Название |
|----|----------|
| FR-10–15 | Запись, выбор инструментов, лимиты |

## Критерии приёмки

| ID | Критерий |
|----|----------|
| AC-001 | Выбор до 3 мест с учётом `max_seats` |
| AC-002 | Прокат disabled при недостатке наборов |
| AC-003 | Цена отображается из `price_total` API |
| AC-004 | При 409/410 — понятная ошибка и переход |
