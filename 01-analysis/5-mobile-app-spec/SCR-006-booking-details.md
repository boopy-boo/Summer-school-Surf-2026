# Детали брони + отмена

**ID:** SCR-006  
**Тип:** Экран  
**Домен:** 03. Бронирования  
**Приоритет:** Critical  
**Зона:** АЗ

## Обзор

Полная информация о брони с возможностью отмены (если `start_at` в будущем). Показывает статус, детали, правило отмены.

## Навигация

| Вход | Выход |
|------|-------|
| SCR-005 (тап по брони) | BS-003 (тап «Отменить») |

## Используемые запросы

### getBooking

**GET `/bookings/{bookingId}`**

| Результат | Условие | UI-реакция |
|-----------|---------|------------|
| Успех | 200 | Детали брони |
| 404 | Не найдена | Error state |

### cancelBooking

**DELETE `/bookings/{bookingId}`**

| Результат | Условие | UI-реакция |
|-----------|---------|------------|
| Успех (ранняя) | 200, `status=cancelled` | Снек «Бронь отменена» |
| Успех (поздняя) | 200, `status=late_cancel` | Снек «Поздняя отмена» |
| 403 | Уже отменена | «Бронь уже отменена» |
| 403 | Занятие началось | «Нельзя отменить прошедшее» |

## Состояния CTA

| Условие | CTA |
|---------|-----|
| `start_at` в будущем + `status=active` | «Отменить бронь», активна |
| `start_at` в прошлом | Неактивна или скрыта |
| `status != active` | Неактивна |

## Критерии приёмки

| ID | Критерий |
|----|----------|
| AC-001 | Показываются все детали брони |
| AC-002 | Отмена доступна только для предстоящих активных броней |
| AC-003 | При ранней отмене — снек «Бронь отменена», места возвращены |
| AC-004 | При поздней отмене — снек «Поздняя отмена», места не освобождены |
