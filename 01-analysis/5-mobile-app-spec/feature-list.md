# Фича-лист клиентского приложения «Глина»

> Перечень функций, приоритетов и трассировка до требований.

## Фичи по приоритету

### Critical (MVP — запуск)

| # | Фича | Экран | Требования | Описание |
|:--|:-----|:------|:-----------|:---------|
| 1 | Регистрация / Вход | SCR-001 | FR-1, FR-2, FR-43 | SMS OTP: телефон → код → имя для новых. |
| 2 | Список слотов | SCR-002 | FR-9, FR-38 | Список занятий на 7 дней, группировка по дате. |
| 3 | Фильтры слотов | BS-001 | FR-38 | По дате, программе (лепка/круг), мастеру. |
| 4 | Карточка слота | SCR-003 | FR-9a, FR-44 | Детали: программа, мастер, адрес, карта, цена. |
| 5 | Оформление записи | SCR-004 | FR-10–15 | Выбор мест (1–3), инструментов, подтверждение. |
| 6 | Подтверждение записи | BS-002 | FR-10 | Шторка успеха + запрос push-разрешения (первая запись). |
| 7 | Мои бронирования | SCR-005 | FR-35a | История с пагинацией, разделение предстоящие/прошедшие. |
| 8 | Детали брони + отмена | SCR-006 | FR-16–18 | Просмотр брони, кнопка отмены, статус. |
| 9 | Подтверждение отмены | BS-003 | FR-16–18 | Шторка подтверждения с объяснением правила 2 часов. |
| 10 | Профиль клиента | SCR-007 | FR-47–49 | Просмотр/редактирование имени, выход, удаление. |

## Трассировка: UC → Экран → API

| UC | Экраны | API |
|:--|:-------|:----|
| UC-1 Регистрация | SCR-001 | POST /auth/otp/send, POST /auth/otp/verify |
| UC-2 Просмотр слотов | SCR-002, BS-001 | GET /slots, GET /masters |
| UC-3 Запись | SCR-003, SCR-004, BS-002 | GET /slots/{id}, POST /bookings |
| UC-4 Отмена | SCR-005, SCR-006, BS-003 | GET /bookings, DELETE /bookings/{id} |
| UC-5 История | SCR-005, SCR-006 | GET /bookings |
| UC-6 Push | — | Push-токен регистрируется на устройстве |

## Ключевые метрики запуска

| Метрика | Цель |
|:--------|:-----|
| Доля онлайн-записей | ≥ 70% через месяц |
| Время на запись | ≤ 2 мин (vs ~15 мин ручной) |
| 0 двойных броней | После запуска |
