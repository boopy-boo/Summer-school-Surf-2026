# Дизайн-макеты «Гончарная мастерская «Глина»

## Базовая ссылка на прототип

```
pottery-workshop/02-development/ui-prototype/index.html
```

Открыть в браузере: `start pottery-workshop/02-development/ui-prototype/index.html`

---

## Карта экранов

| # | Экран | Якорь | API Endpoints | Описание |
|---|-------|-------|---------------|----------|
| 1 | **Вход по телефону** | `index.html#login` | `POST /auth/otp/send` | Экран авторизации, ввод номера телефона |
| 2 | **Ввод SMS-кода** | `index.html#otp` | `POST /auth/otp/verify` | Подтверждение OTP, таймер повторной отправки |
| 3 | **Расписание слотов** | `index.html#slots` | `GET /slots`, `GET /masters` | Список мастер-классов с фильтрами и карточками |
| 4 | **Оформление брони** | `index.html#booking` | `POST /bookings` | Выбор участников, аренда круга, калькулятор цены |
| 5 | **Мои записи** | `index.html#bookings` | `GET /bookings`, `DELETE /bookings/{id}` | История бронирований, статус, отмена |

---

## Навигационный поток (User Flow)

```
[SCR-001] Вход по телефону
    → POST /auth/otp/send
    → [SCR-001b] Ввод SMS-кода
        → POST /auth/otp/verify
        → [SCR-002] Расписание слотов
            
            ┌─→ [SCR-003] Карточка слота (клик по карточке)
            │   → [SCR-004] Оформление брони
            │       → POST /bookings
            │       → [BS-002] Успешная запись
            │
            ├─→ [BS-001] Фильтры (иконка фильтра)
            │
            └─→ [SCR-005] Мои записи (таб-бар)
                → GET /bookings
                → [SCR-006] Детали брони (клик по записи)
                    → DELETE /bookings/{id} → [BS-003] Подтверждение отмены
                → [SCR-007] Профиль
```

## Список дизайн-экранов → API-сопоставление

### SCR-001 — Регистрация / Авторизация
- **Файл прототипа:** `index.html` (экран `login-screen`)
- **API:** `POST /auth/otp/send` → `POST /auth/otp/verify`
- **Состояния:**
  - Поле ввода номера, маска +7
  - Кнопка «Получить код» (disabled пока номер не валиден)
  - Ошибка: «Неверный номер телефона»

### SCR-001b — Ввод OTP
- **Файл прототипа:** `index.html` (экран `otp-screen`)
- **API:** `POST /auth/otp/verify`
- **Состояния:**
  - 6 цифр, автофокус
  - Таймер «Отправить повторно через 45 сек»
  - Ошибка: «Неверный код» (3 попытки)

### SCR-002 — Список слотов
- **Файл прототипа:** `index.html` (экран `slots-screen`)
- **API:** `GET /slots?date_from=&date_to=&program_type=&limit=`
- **Состояния:**
  - Табы: Все / Ручная лепка / Гончарный круг
  - Карточка слота: дата, мастер, цена, остаток мест
  - Пустое состояние: «Нет доступных занятий»
  - Pull-to-refresh
  - Бесконечный скролл (pagination)

### SCR-004 — Оформление брони
- **Файл прототипа:** `index.html` (экран `booking-screen`)
- **API:** `POST /bookings`
- **Состояния:**
  - Выбор 1–3 участников (select)
  - Чекбокс «Нужен гончарный круг» (rental)
  - Калькулятор: сумма = price × seats + rental_price × rental_count
  - Кнопка «Подтвердить» (disabled если слот filled/cancelled)
  - Ошибка 409: «Мест нет»
  - Ошибка 410: «Занятие отменено мастерской»

### SCR-005 — Мои записи
- **Файл прототипа:** `index.html` (экран `bookings-screen`)
- **API:** `GET /bookings`, `DELETE /bookings/{id}`
- **Состояния:**
  - Список активных броней (зелёный статус)
  - Архив: отменённые / прошедшие (серый, opacity 0.7)
  - Кнопка «Отменить запись до {time}» (показывать, если >2 часов до старта)
  - Bottom nav: Поиск / Мои записи / Профиль

### BS-001 — Фильтры (bottom sheet)
- **API:** `GET /slots?date_from=&date_to=&program_type=&master_id=`
- **Элементы:**
  - Диапазон дат (date picker)
  - Тип программы: checkbox [handbuilding, wheel]
  - Мастер: dropdown со списком `GET /masters`
  - Кнопки: Применить / Сбросить

### BS-002 — Успешная запись (modal)
- **API:** ответ `201` на `POST /bookings`
- **Элементы:**
  - Иконка ✓, заголовок «Вы записаны!»
  - Детали брони: дата, адрес, мастер
  - Кнопка «Добавить в календарь»
  - Кнопка «Перейти к моим записям»

### BS-003 — Подтверждение отмены (dialog)
- **API:** `DELETE /bookings/{id}`
- **Элементы:**
  - Заголовок: «Отменить запись?»
  - Текст: «Вернём деньги в течение 3–5 рабочих дней»
  - Кнопки: «Да, отменить» / «Нет, оставить»
  - После отмены: возврат к списку, архив
